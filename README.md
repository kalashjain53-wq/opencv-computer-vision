# OpenCV Computer Vision

A collection of Computer Vision projects built while learning OpenCV using Python.

The goal of this repository is to understand the fundamentals of Computer Vision before moving on to modern Deep Learning approaches.

---

## Topics Covered

- Reading Images & Videos
- Drawing on Images
- Image Transformations
- Color Spaces
- Image Splitting & Merging
- Bitwise Operations
- Masking
- Blurring
- Thresholding
- Histograms
- Gradients
- Canny Edge Detection
- Contours
- Haar Cascade Face Detection
- LBPH Face Recognition
